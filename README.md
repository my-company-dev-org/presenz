# PRESENZ Landing Page

A modern landing page for PRESENZ - an app that brings together live art, music, fashion, cinema, and city energy into one living map visible only while it's happening.

## Features

- Modern hero section with gradient backgrounds
- Event discovery cards with live indicators
- iPhone mockups showcasing the app interface
- Category-based exploration section
- Responsive design with Tailwind CSS
- Smooth animations and transitions
- Footer with social media links

## Tech Stack

- **Framework**: Next.js 14+ with App Router
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Package Manager**: npm

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser
